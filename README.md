# Bellman-Ford

Формат входных данных:
        Первая строка: 
                        * кол-во вершин
                        * кол-во рёбер
                        * номер начальной вершины
                        * номер конечной вершины
        Следующие строки:
                        * номер вершины a
                        * номер вершины b
                        * цена пути из a в b

Все значения в ряду разделены запятыми (самопальный CSV).
Не оставляйте пустую линию в конце, парсеру от этого плохеет.
